Alasan nya karena dengan semantic HTML SEO(Search Engine Optimation) lebih mudah memahami alur program yang kita buat, karena tag semantic HTML memiliki makna struktur code yang jelas 
contoh tag semantic HTML yang saya pakai :
<nav>, <section> <footer> <article>
contoh tag non semantic HTML yang saya pakai
<div>
