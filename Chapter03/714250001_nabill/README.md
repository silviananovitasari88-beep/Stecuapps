
Saya membuat website e-commerce penjualan hoodie lokal secara online bernama “Bielsstore”. Website ini menampilkan berbagai produk hoodie dari tiga brand lokal, yaitu AETN.CO, CHMB (CHAMBRE), dan CALLE DE LARACHE, masing-masing dengan 6 produk.
Pengguna dapat melihat semua koleksi hoodie, menambahkan produk ke keranjang belanja, dan melakukan checkout dengan memilih metode pembayaran seperti transfer bank, COD, atau QRIS.


Website ini dibangun menggunakan HTML untuk struktur halaman, CSS untuk tampilan desain, dan JavaScript untuk logika interaktif.
JavaScript digunakan untuk:

a. Menampilkan data produk secara dinamis dari array products.
b. Menambahkan item ke keranjang belanja saat tombol “Tambah ke Keranjang” ditekan.
c. Menghitung total harga otomatis di halaman checkout.
d. Menghapus isi keranjang setelah proses pembayaran selesai.
