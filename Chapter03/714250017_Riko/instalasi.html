Instalasi

Sebelum membuat proyek Laravel pertama Anda, pastikan mesin lokal Anda telah menginstal PHP dan Komposer . Jika Anda mengembangkan di MacOS, PHP dan Komposer dapat diinstal dalam hitungan menit melalui Laravel Herd . Selain itu, kami menyarankan untuk menginstal Node dan NPM . Setelah Anda menginstal PHP dan Composer, Anda dapat membuat proyek Laravel baru melalui perintah Composer create-project:

"composer create-project laravel/laravel example-app"


Atau, Anda dapat membuat proyek Laravel baru dengan menginstal installer Laravel secara global melalui Composer:

composer global require laravel/installer

laravel new example-app


Setelah proyek dibuat, mulai server pengembangan lokal Laravel menggunakan serveperintah Laravel Artisan:

cd example-app

php artisan serve


Setelah Anda memulai server pengembangan Artisan, aplikasi Anda akan dapat diakses di browser web Anda di http://localhost:8000 . Selanjutnya, Anda siap untuk mulai mengambil langkah selanjutnya ke dalam ekosistem Laravel . Tentu saja, Anda mungkin juga ingin mengkonfigurasi database .
