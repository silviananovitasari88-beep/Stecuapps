Proyek ini merupakan implementasi portofolio web personal yang didesain secara responsif dan minimalis , dibangun menggunakan teknologi dasar HTML5 untuk struktur dan CSS3 (Cascading Style Sheets) untuk presentasi visual. Tujuannya adalah untuk menampilkan profil akademis, pengalaman proyek, dan skillset teknis saya secara profesional dan estetis, dengan penekanan pada keterbacaan (readability) dan pengalaman pengguna (UX) yang superior.
External CSS , memisahkan secara ketat antara konten (index.html) dan presentasi (style.css), menjamin arsitektur kode yang bersih, terstruktur, dan mudah dikelola (maintainable). Sementara Internal CSS adalah kebalikan dari External CSS
