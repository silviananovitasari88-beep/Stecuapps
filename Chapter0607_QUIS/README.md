# Soal A — NPM GANJIL
## Landing Page “Event Festival Musik”
### Music Festival Landing Page (Colorful & Energetic)
1. Buat sebuah landing page untuk mempromosikan Festival Musik 2025 (bebas nama dan tema).
2. Syarat Utama:
- Gunakan HTML, CSS, JavaScript.
- Wajib ada:
- Hero section dengan background bergerak (CSS animation atau video).
- Jadwal acara (lineup) dalam bentuk accordion atau tabs (JavaScript).
- Daftar performer dengan card foto.
- Tiket (harga + CTA “Buy Tickets”).
- Footer sosial media.
3. Desain harus colorful, energik, dan responsif.
4. Tambahkan efek kecil: hover animasi, scroll reveal, atau efek neon.
5. Output:
- Lakukan Pull Request dan Push ke Repository Github ini
- Untuk Adlink Proyek lengkap + screenshot tampilan desktop & mobile masukan kedalam 1 file PDF dan kirimkan untuk dilakukan pengujian Coding di pertemuan berikutnya


# Soal B — NPM GENAP
## Landing Page “Online Course Platform”
### EduLearn Landing Page (Clean & Modern)
1. Buat landing page untuk platform kursus online (mirip Udemy/Coursera versi sederhana).
2. Syarat Utama:
- Gunakan HTML, CSS, JavaScript.
- Wajib ada:
- Hero dengan ilustrasi atau mockup aplikasi.

3. Desain modern, minimalis, dan responsif.
4. Tambahkan animasi halus (hover, fade-in, atau slide).
5. Responsif (mobile–tablet–desktop).
6. Output:
- Lakukan Pull Request dan Push ke Repository Github ini
- Untuk Adlink Proyek lengkap + screenshot tampilan desktop & mobile masukan kedalam 1 file PDF dan kirimkan untuk dilakukan pengujian Coding di pertemuan berikutnya
